# Front-EndTechnologiesBasics

This course focuses on expanding the knowledge and practical skills in the area of front-end testing.
The course covers:
-Understanding the fundamentals of HTML and CSS
-Working with JavaScript (AJAX.Promises.Async/Await)
-Manipulating the DOM (Document Object Model)
-Handling events in web applications
-Using Playwright and QUnit for front-end automated testing
-Applying practical testing approaches for different front-end scenarios
-Selecting appropriate tools for front-end test automation
